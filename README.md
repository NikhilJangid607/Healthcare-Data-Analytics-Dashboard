# 🏥 Healthcare Data Analytics Dashboard (Microsoft Excel)

## 📌 Project Overview
This project is an interactive **Healthcare Data Analytics Dashboard** developed using **Microsoft Excel**. The dashboard provides meaningful insights into healthcare billing, medical conditions, insurance providers, hospitals, doctors, and patient admissions through dynamic visualizations and slicers.

It enables users to explore healthcare data efficiently and make data-driven decisions.

---

## 🚀 Dashboard Features

- 💰 Total Billing Amount
- 📊 Average Billing Amount
- 🏥 Number of Hospitals
- 👨‍⚕️ Number of Doctors
- 🩺 Most Common Medical Condition
- 📈 Annual Billing Summary (2019–2024)
- 🥧 Insurance Provider-wise Billing Analysis
- 📉 Medical Condition Analysis by Admission Type
- 📌 Distribution of Medical Conditions Across Blood Types
- 🎛 Interactive Slicers for Dynamic Filtering
